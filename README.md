# rust-osinfor

## 简介
Rust开发的系统工具，使用RustRover作为Ide。

## 功能列表
- [x] 获取系统架构;
- [x] 展示本工具的使用方法;
- [ ] 获取系统主机名；
- [ ] 修改系统主机名；
- [ ] 获取系统软件包数量；
- [ ] 获取系统硬件信息(机器型号，bios信息等);
- [ ] 磁盘使用情况或者格式化磁盘；
- [ ] 获取系统ip;
- [ ] 获取系统版本；
- [ ] 修改管理员密码；
